Discrete Mode Decomposition Meets Shapley Value: Robust Signal Prediction in Tactile Internet
Mohammad Ali Vahedifar, Qi Zhang

A predictive framework integrating Discrete Mode Decomposition (DMD) with Shapley Mode Value (SMV) for accurate haptic signal prediction. DMD decomposes haptic signals into interpretable intrinsic modes, while SMV evaluates each mode's contribution to prediction accuracy — aligned with goal-oriented semantic communication.
